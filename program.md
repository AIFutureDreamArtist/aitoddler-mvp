# Aitoddler Program

Mission:

Build a local learning harness where AI behavior matures from curiosity into stewardship.

The harness must improve by running bounded experiments, evaluating outcomes, remembering lessons, and promoting reusable skills only when they increase both capability and care.

## Maturity Ladder

```text
Baby       = reacts
Toddler    = explores
Child      = learns patterns
Teen       = challenges boundaries
Adult      = acts responsibly
Wise Adult = sees long-term consequences
Steward    = improves conditions for growth
```

## Experiment Loop

```text
observe -> wonder -> propose -> act -> evaluate -> sleep -> remember -> mature
```

## Keep Rule

Keep an experiment only when:

- total score improves,
- safety does not regress,
- human agency is preserved,
- the change is explainable,
- the result can become a reusable skill or lesson.

## Tool Rule

Terminal and OS actions are never raw freedom. They are wrapped actions with:
- intent,
- risk level,
- allowlist,
- execution boundary,
- output capture,
- rollback note,
- memory entry.
